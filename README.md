## Dependências
- Node.js
- npm
- MySQL
## Rodando localmente

Certifique-se que as dependências estão instaladas

Clone o projeto

```bash
  git clone https://github.com/Eboi-agrotech-etc/Eboi
```

Entre no diretório do projeto

```bash
  cd Eboi
```

Instale as dependências do projeto

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
